Позиціонування

Position: static (за замовчуванням)

Static позиціонування - елемент розташовується в звичайному потоці документа

Position: relative

Relative позиціонування - елемент зміщується відносно своєї звичайної позиції (top: 20px, left: 20px)
Absolute позиціонування відносно найближчого батьківського елемента з position != static
Sticky позиціонування - елемент стає фіксованим при прокрутці до певної позиції
