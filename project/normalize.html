🔧 Normalize CSS

Normalize.css делает стили более унифицированными в разных браузерах, сохраняя полезные настройки по умолчанию.

Что такое Normalize.css?

Normalize.css - это современная HTML5-готовая альтернатива традиционному CSS reset. Она:

  • Сохраняет полезные настройки по умолчанию вместо их стирания
  • Нормализует стили для широкого спектра элементов
  • Исправляет ошибки и общие несовместимости браузеров
  • Улучшает юзабилити с помощью тонких улучшений
  • Объясняет, что делает код, используя детальные комментарии

Основные принципы

Сохранение полезных настроек

В отличие от CSS reset, normalize.css не убирает все стили, а сохраняет полезные настройки браузера.

Нормализация стилей

Обеспечивает единообразное отображение элементов во всех браузерах.

Исправление ошибок

Устраняет распространенные ошибки отображения в различных браузерах.

Пример кода Normalize.css

/*! normalize.css v8.0.1 | MIT License | github.com/necolas/normalize.css */

/* Document
   ========================================================================== */

/**
 * 1. Correct the line height in all browsers.
 * 2. Prevent adjustments of font size after orientation changes in iOS.
 */

html {
  line-height: 1.15; /* 1 */
  -webkit-text-size-adjust: 100%; /* 2 */
}

/* Sections
   ========================================================================== */

/**
 * Remove the margin in all browsers.
 */

body {
  margin: 0;
}

/**
 * Render the `main` element consistently in IE.
 */

main {
  display: block;
}

Сравнение с CSS Reset

Аспект Normalize.css CSS Reset
Подход Сохраняет полезные настройки Обнуляет все стили
Размер Больше (~7KB) Меньше (~1KB)
Документация Подробные комментарии Минимальные комментарии
Поддержка HTML5 элементы Основные элементы
