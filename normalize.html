Normalize CSS

Що таке Normalize.css?

Normalize.css робить стилі більш уніфікованими в різних браузерах, зберігаючи корисні налаштування за замовчуванням.

Normalize.css зберігає корисні стилі за замовчуванням, на відміну від CSS Reset, який видаляє їх повністю.

Основні принципи

  • Зберігає корисні стилі за замовчуванням
  • Нормалізує стилі для широкого спектру елементів
  • Виправляє помилки та несумісності браузерів
  • Пояснює код детальними коментарями

Приклад нормалізації

html { line-height: 1.15; /* Покращує читабельність */ -webkit-text-size-adjust: 100%; /* iOS Safari */ } body { margin: 0; /* Видаляє стандартні відступи */ }
