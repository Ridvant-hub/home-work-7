# 🎨 CSS Демонстрації - Навчальний сайт

Інтерактивний навчальний сайт для вивчення основних CSS концепцій та властивостей.

## 📋 Особливості

- **Інтерактивні демонстрації** CSS властивостей
- **Сучасний дизайн** з градієнтами та анімаціями
- **Адаптивна верстка** для всіх пристроїв
- **Плавна навігація** з sticky меню
- **Progress bar** для відстеження прокрутки
- **Back to top** кнопка для зручності

## 🗂️ Структура сайту

### Основні сторінки:

1. **index.html** - Головна сторінка з блочними та інлайновими елементами
2. **display.html** - Демонстрація display властивостей (block, inline, flex, grid)
3. **positioning.html** - Типи позиціонування (static, relative, absolute, fixed, sticky)
4. **float-clear.html** - Плаваючі елементи та їх очищення
5. **overflow.html** - Керування переповненням контенту
6. **normalize.html** - Пояснення Normalize.css
7. **reset.html** - Пояснення CSS Reset

### CSS файли:

- **css/style.css** - Основні стилі сайту
- **css/normalize.css** - Normalize.css для кросбраузерної сумісності
- **css/reset.css** - CSS Reset для повного контролю

## 🚀 Як запустити

1. Відкрийте `index.html` у браузері
2. Використовуйте навігацію для переходу між сторінками
3. Експериментуйте з інтерактивними елементами

## 🎯 Навчальні цілі

Цей сайт допомагає зрозуміти:

- Різницю між блочними та інлайновими елементами
- Різні типи display властивостей
- Як працює позиціонування в CSS
- Коли використовувати float та чому Flexbox краще
- Як керувати переповненням контенту
- Різницю між Normalize.css та CSS Reset

## 🛠️ Технології

- **HTML5** - семантична розмітка
- **CSS3** - сучасні стилі з градієнтами, анімаціями та Grid/Flexbox
- **JavaScript** - інтерактивність та навігація
- **Адаптивний дизайн** - Media queries для мобільних пристроїв
