Reset CSS

Що таке CSS Reset?

CSS Reset видаляє всі стилі браузера за замовчуванням, надаючи чисту основу для створення власних стилів.

CSS Reset видаляє ВСІ стилі, включаючи корисні. Це означає, що вам потрібно буде стилізувати кожен елемент самостійно.

Основні принципи

  • Скидає всі margin та padding до 0
  • Видаляє стилі списків
  • Скидає стилі посилань
  • Уніфікує шрифти та розміри

Приклад reset коду

* { margin: 0; padding: 0; box-sizing: border-box; } ul, ol { list-style: none; } a { text-decoration: none; color: inherit; }

Порівняння з Normalize

CSS Reset

  • ❌ Видаляє ВСІ стилі
  • ✅ Чиста основа
  • ❌ Потребує більше роботи

Normalize CSS

  • ✅ Зберігає корисні стилі
  • ✅ Менше роботи
  • ✅ Краща семантика
Вибирайте між Reset та Normalize залежно від ваших потреб. Для швидкого прототипування краще Normalize, для повного контролю - Reset.
