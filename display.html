Свойство display определяет, как элемент отображается на странице. Это одно из самых важных CSS свойств для создания макетов.

Box-sizing Property

Свойство box-sizing определяет, как рассчитывается общая ширина и высота элемента.

Content-box (по умолчанию)

Ширина и высота применяются только к содержимому. Padding и border добавляются сверх указанных размеров.

content-box
Общая ширина: 250px
width: 200px + padding: 40px + border: 10px = 250px общая ширина

Border-box

Ширина и высота включают содержимое, padding и border. Размер содержимого уменьшается автоматически.

border-box
Общая ширина: 200px
width: 200px (включает content + padding + border)
